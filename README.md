# SyncContainers
Sync 2 containers (vector & map) with multipthreading

Task: Sync 2 containers vector(int) & map(int)(int). Duplicated items don't remove.

Example:

Vector: 9,9,0,3,7,6,5,5

Map: 1,2,0,9,5,7,3,5

Result: 9, 0, 3, 7, 5, 5 - in both containers.
